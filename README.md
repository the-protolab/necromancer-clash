# Necromancer's Clash

A single-file HTML match-3 battle game made for a game jam on the theme of fighting your own ghost. Drag through three or more touching blocks of the same kind — in a line, an L, or a zig-zag — to attack, heal, shield, or freeze your opponent. On your first run you face a wild wraith; win and your run is recorded as the ghost you must out-play next time. Everything — structure, style, and logic — lives in `code.html`, with no build step and no dependencies. Open `code.html` in any browser to play.
